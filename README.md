# GithubActionsDemo

This repository is a hands‑on exploration of **GitHub Actions** — GitHub’s native CI/CD (Continuous Integration and Continuous Deployment) platform.  
The goal is to learn how workflows are defined, triggered, and executed to automate tasks such as testing, building, and deploying code.

---

## 📚 What I’m Learning
- How to create and configure **workflow files** (`.github/workflows/*.yml`)
- Understanding **events** (push, pull_request, schedule, etc.)
- Using **jobs** and **steps** to structure automation
- Running tasks on different **runners** (Ubuntu, Windows, macOS)
- Leveraging **actions** from the GitHub Marketplace
- Passing **secrets** and environment variables securely
- Building simple pipelines for testing and deployment

## Goals of This Project
- Practice writing and debugging workflows
- Understand how automation fits into modern software development
- Build confidence in using CI/CD pipelines
- Prepare for real‑world projects where GitHub Actions can save time and reduce manual effort
